# Консольное приложение для управления списком задач
Функционал приложения:
1. Добавление/удаление задач;
2. Вывод всех задач;
3. Удаление всех выполненных задач;
4. Изменения файла для хранения информации;
5. Вывод статистики по выполнению задач;
6. Изменение статуса задач на выполнено/невыполненное 

Структура приложения:
Вся программа разбита на 3 глобальных модуля:
1. Работа с текстовым файлом(Data_Base);
2. Работа с выводом и вводом в консоль(интерфейс);
3. Запуск программы и кейс меню(main).

Структура текстового файла:
1. Первая строчка файла выделена на техническую информацию(последний созданный id);
2. Далее идут строчки с записями задач. Одна строчка - одна задача;
3. Каждая строчка хранит в себе id, заголовок, текстовое содержание, срок, статус выполнения задачи;
4. Все пункты разделены специальным символом \x1F(Ну чтоб пользователь точно его ввести не мог);
5. Каждая строка одинакового размера(пустота заполняется пробелами).

Интерфейс приложения:
1. Консоль, навигация по которой осуществляется через стрелки на клавиатура(вверх, вниз),
   также используются кнопки ESC для выхода из приложения и ENTER для выбора пункта меню;
2. Присутствует цветной шрифт :)
3. Присутствует защита от некорректного ввода, и возможность отмены ввода;

Технические особенности:
1. Язык: C++
2. Библиотеки: стандартные;
3. Типизация: Строгая статическая.
